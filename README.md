## 物流配送管理系统
### 这是我的一次webgis二次开发作业。
### 开发者：loyoung
### 开发时间：2018/03
### 语言：html、css、JavaScript、jsp.
### 数据库：mysql.
### 环境：win10、eclipse、sublime.
### 受众：快递公司管理人员
### 系统功能：  
    1.登录；   
    2.实现快递单号增删改、查询、储存、管理、可视化；   
    3.分析出快递配送员到指定某地的最短路径、快递员一次性能送最多快递的路径（未实现）、在指定多个快递点时实现多点最优路径、缓冲分析、导航；   
    4.热力图，流向图，轨迹图。
